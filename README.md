
The Interactive Age Calculator is a simple web application developed using HTML, CSS, and JavaScript. It allows users to calculate their exact age based on their date of birth. The application displays the age in years, months, and days and also shows the total time lived in hours, minutes, and seconds.

## Features

- Calculate exact age (Years, Months, Days)
- Show total days, hours, minutes, and seconds lived
- Live age updating feature
- Next birthday countdown
- Simple and interactive user interface

## Technologies Used

- HTML
- CSS
- JavaScript
